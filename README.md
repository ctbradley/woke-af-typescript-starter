# woke-af-typescript-starter

> A Woke AF TypeScript starter project.

## Getting Started

The easiest way to get started is to run:

```sh
git clone https://github.com/ctbradley/woke-af-typescript-starter.git
cd woke-af-typescript-starter
npm i
npm run compile
npm run start
```
